# Centrum Pomocy Profesjonalnej

## Project Overview
Centrum Pomocy Profesjonalnej is a web application aimed at providing psychological help and support for individuals facing various life challenges. The platform connects users with qualified psychologists, therapists, and mental health professionals in Poland. The website offers information about services, a knowledge base, and the ability to book appointments for consultations.

## Installation
To install and set up this project, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
   
2. Navigate to the project directory:
   ```bash
   cd <project-directory>
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage
The application can be accessed via a web browser at `http://localhost:3000`. Users can navigate the site to learn about various mental health services, read informative articles from the knowledge base, and book appointments online.

## Features
- **User-Friendly Interface:** Clean, intuitive design for easy navigation.
- **Information Access:** A comprehensive repository of articles on mental health topics.
- **Appointment Booking:** Users can easily schedule their consultations online.
- **Multi-Language Support:** Offers content in Polish and English.
- **Professional Team:** Access to experienced therapists and psychologists.

## Dependencies
The application relies on several npm packages. Below are the primary dependencies specified in `package.json`:

- **React**: A JavaScript library for building user interfaces.
- **Next.js**: A React framework for production, enabling server-side rendering and static site generation.
- **Swiper**: A library for creating beautiful sliders and carousels.
- **CSS Modules**: For modular and scoped CSS styling.

For a complete list of dependencies, refer to the `package.json` file in the root directory.
